# Notes linguistiques

Ce dossier documente l'apprentissage du grec ancien à travers *Les Perses*.

## Structure

- **grammaire-et-morphologie.md** : Cas, déclinaisons, conjugaisons, syntaxe
- **vocabulaire-cle.md** : Mots importants, hapax, termes perses transcrits en grec
- **notes-metriques.md** : Formes contractées, licences poétiques, dialectes (dorien, éolien)

## Principe

Ces notes documentent les **difficultés rencontrées** et les **découvertes linguistiques**, pas une grammaire grecque complète. Ce n'est pas un manuel, c'est un journal d'apprentissage ciblé sur cette pièce.

## Niveau de confiance

Chaque note est étiquetée selon le niveau de confiance :
- 🟢 **Confiance haute** : Vérifié dans une grammaire de référence ou confirmé par un helléniste
- 🟡 **Confiance moyenne** : Hypothèse plausible mais non vérifiée
- 🔴 **Confiance basse** : Question ouverte, blocage, demande de vérification externe
