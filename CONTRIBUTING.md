# Comment contribuer

Ce projet documente un parcours d'apprentissage personnel sur *Les Perses* d'Eschyle. Mais il est ouvert aux contributions externes.

## Types de contributions acceptées

### 1. Corrections factuelles

Si vous repérez une erreur (traduction incorrecte, référence historique fausse, analyse grammaticale erronée), ouvrez une **Issue** avec :
- Le fichier et la ligne concernés
- L'erreur identifiée
- La correction proposée avec source

### 2. Propositions d'interprétation alternatives

Si vous avez une lecture différente d'un passage, deux options :

**Option A : Débat via Issue**  
Ouvrez une Issue avec :
- Le passage concerné (numéro de vers)
- Votre interprétation
- Les arguments qui la soutiennent
- Les objections possibles à votre propre lecture

**Option B : Pull Request**  
Proposez une modification du commentaire existant en :
- Créant une branche dédiée
- Ajoutant votre interprétation en note ou en section alternative
- Expliquant pourquoi votre lecture est préférable ou complémentaire

### 3. Ajout de traductions

Si vous souhaitez ajouter une traduction qui n'est pas encore dans le dépôt :
- Vérifiez qu'elle est dans le domaine public ou que vous avez l'autorisation
- Créez un fichier dans `traductions/` avec le format : `auteur-annee.md`
- Indiquez la source complète en en-tête du fichier

### 4. Ressources audiovisuelles

Si vous connaissez une captation, une conférence, un cours accessible en ligne sur *Les Perses*, ajoutez-le dans `ressources-audiovisuelles/` avec :
- Lien vers la ressource
- Date et contexte (mise en scène, intervenant, lieu)
- Brève description (2-3 phrases)

## Ce qui n'est PAS accepté

- Modifications du journal de lecture (c'est un document personnel, immutable)
- Ajout de contenu hors sujet (autres pièces d'Eschyle, contexte général sur la Grèce antique sans lien direct avec *Les Perses*)
- Traductions automatiques ou non vérifiées
- Contenus sous copyright sans autorisation

## Niveau de confiance

Quand vous proposez une contribution, indiquez votre niveau de confiance :

- 🟢 **Confiance haute** : Vous êtes helléniste, ou vous vous basez sur des sources académiques vérifiées
- 🟡 **Confiance moyenne** : Vous avez une hypothèse solide mais non vérifiée par des pairs
- 🔴 **Confiance basse** : Vous posez une question ou proposez une piste exploratoire

Cela permet de distinguer les contributions établies des pistes de recherche.

## Tone et méthode

Ce projet applique le principe du **steelmanning** : avant de critiquer une interprétation, reformulez-la dans sa version la plus forte et la plus charitable.

Évitez :
- Les arguments d'autorité ("X dit que donc c'est vrai")
- Les caricatures d'interprétations adverses
- Les jugements moraux sur les lectures alternatives

Privilégiez :
- Les arguments textuels (références précises aux vers)
- Les comparaisons avec d'autres passages de la pièce
- Les objections sérieuses à votre propre thèse

## Questions

Pour toute question sur le projet, ouvrez une Issue avec le label `question`.
