# Commentaire ligne à ligne

Ce dossier contient l'analyse progressive des *Perses* d'Eschyle, section par section, dans l'ordre dramaturgique de la pièce.

## Méthode de commentaire

Chaque section contient :

- Le texte grec (quand disponible)
- La traduction française (Mazon comme référence)
- L'analyse dramaturgique (fonction de la scène dans l'économie de la pièce)
- Les notes linguistiques (difficultés grammaticales, vocabulaire clé)
- Les hypothèses interprétatives (politique, religieuse, psychanalytique)
- Le niveau de confiance (🟢🟡🔴)

## Progression

Les fichiers seront remplis progressivement, au fil de la lecture et de l'étude. L'hétérogénéité est assumée : certaines sections seront très détaillées, d'autres resteront provisoires.

## Constats transversaux

Les analyses ont fait émerger plusieurs constats lexicaux vérifiables :

- **Inventaire de ζυγ-** : 5 occurrences dans la pièce, toutes dans les registres lyrique/onirique (Chœur, Atossa en rêve), aucune dans les registres factuels (Messager, Xerxès) → voir `notes-linguistiques/inventaire-zygon.md`
- **Vocabulaire de la faute cosmique** (ζυγόν, πέδαι, Poséidon, Hellespont) : réservé à des régimes énonciatifs spécifiques, voir `REVISION_THESE.md`
- **Silence du Messager sur le pont** : absence totale de mention de l'Hellespont dans le récit de Salamine (v. 249-514)
