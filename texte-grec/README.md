# Texte grec des Perses

Ce dossier contiendra le texte grec complet d'Eschyle, numéroté vers par vers.

## Source

Le texte sera saisi à partir de l'édition **Paul Mazon (1920, Les Belles Lettres)**, qui reprend l'édition critique de référence.

## Format

Chaque vers sera numéroté et présenté dans un format permettant l'annotation progressive :

```
1. [Texte grec du vers 1]
2. [Texte grec du vers 2]
...
```

## Annotations

Au fil de l'apprentissage, des annotations seront ajoutées :
- Analyse morphologique des formes difficiles
- Identification des licences poétiques et contractions métriques
- Notes sur les dialectes (dorien, éolien) dans les parties chorales
- Références croisées avec le commentaire ligne à ligne

## Progression

Le texte sera saisi progressivement, au rythme de la lecture et de l'étude. Priorité aux sections dialoguées (plus accessibles) avant les sections lyriques (plus complexes).
